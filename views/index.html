Exercise tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

GET users's exercise log: GET /api/exercise/log?{username}[&from][&to][&limit]

Example: https://granite-psychology.glitch.me/api/exercise/log?username=rajatkn&limit=3&from=06-08-2018&to=06-30-2018
Example Output: 

{
  "username":"rajatkn",
  "from":"Fri Jun 08 2018",
  "to":"Sat Jun 30 2018",
  "limit":"3",
  "exercises":[
  {
    "description":"bowling",
    "duration":"30",
    "date":"Sat Jun 16 2018"
  },
  {
    "description":"yoga",
    "duration":"30",
    "date":"Sat Jun 16 2018"
  },
  {
    "description":"Yoga",
    "duration":"30",
    "date":"Mon Jun 18 2018"
  }]
}

{ } = required, [ ] = optional

from, to = dates (mm-dd-yyyy); limit = number
